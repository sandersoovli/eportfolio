Tagurpidiklassiruum App

The goal of this four-member team project was to create an educational mobile application designed to help users expand their Estonian vocabulary. The app works offline, displaying word definitions for users to guess the corresponding word. All statistics and user settings are stored in local storage.

GitHub (Collaborative Repo): github.com/aleksandertap/tagurpidiklassiruum
Project documentation: Documentation

My role in the team focused primarily on user interface design and user experience optimization, ensuring the app was simple and intuitive. Additionally, I implemented a confetti function in the app that activates when a student answers correctly. The app remembers the last word, allowing users to continue from where they left off. I added buttons with icons in the top corner of the app: on the left are settings and on the right is feedback, which shows how many words the user answered correctly and incorrectly. This helps users track their progress and use the app more conveniently.
